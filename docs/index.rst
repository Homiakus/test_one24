Быстрый старт
=============

Приложение управления устройством - это Python приложение с графическим интерфейсом,
построенное на PySide6, для управления лабораторным оборудованием через Serial соединение.

Основные возможности:

* Управление устройством через Serial порт
* Выполнение команд и последовательностей
* Графический интерфейс с современным дизайном
* Система навигации между страницами
* Dependency Injection контейнер
* Логирование и обработка ошибок

Установка
---------

.. code-block:: bash

   pip install -r requirements.txt

Запуск
------

.. code-block:: bash

   python main.py

Архитектура
===========

Приложение построено по модульной архитектуре с использованием:

* **Core** - основная бизнес-логика
* **UI** - пользовательский интерфейс
* **Utils** - вспомогательные утилиты
* **Config** - управление конфигурацией

Все модули используют строгую типизацию и следуют принципам SOLID.
