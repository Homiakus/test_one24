# Панель управления стендом

Приложение для управления устройствами через Serial-порт с графическим интерфейсом.

## Возможности

- Подключение к устройствам через Serial-порт
- Отправка команд и получение ответов
- Запуск последовательностей команд
- Загрузка прошивки на устройства через PlatformIO
- Автоматическое обновление приложения из репозитория
- Полноэкранный режим работы (F11)
- Настраиваемые кнопки команд и последовательности

## Требования

- Python 3.9 или выше
- PyQt5
- Qt-material
- PySerial
- PlatformIO (для загрузки прошивки)
- Ruff (для проверки кода)

## Разработка

### Использование линтера Ruff

Проект использует линтер [Ruff](https://github.com/astral-sh/ruff) для контроля качества кода. 

Проверка кода:
```bash
ruff check main.py
```

Автоматическое исправление простых ошибок:
```bash
ruff check main.py --fix
```

### Конфигурация

Основные настройки приложения хранятся в следующих файлах:
- `config.toml` - конфигурация кнопок и последовательностей команд
- `serial_settings.json` - настройки Serial-порта
- `update_settings.json` - настройки обновления приложения
- `pyproject.toml` - настройки линтера Ruff
